# sqlite-encrypt-examples


- http://www.sqlite.org/download.html
- https://github.com/utelle/wxsqlite3/releases
- https://github.com/rindeal/SQLite3-Encryption

sqlite version 3.18.0


Download the latest wxsqlite3
Extract the wxsqlite3-*/sqlite3/secure/src dir from the archive to src in the project root dir.


- 直接用 wxsqlite3 中的 sqlite，只要版本新就行
- 编译出来的 sqlite3.dll 直接替换 NavicatforSQLite 中的 sqlite3.dll，可用 NavicatforSQLite 浏览数据
